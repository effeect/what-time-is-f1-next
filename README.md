# What time is F1

This is a static built website with Next.js and it's goal is to do the following :

- Show the timings for the next formula one race
- Allow the user to navigate to different races in the year
- Statically generate each one with Next.JS

# How to run locally

Since it's a basic next.js site, all you need to do is do a `npm install` and `npm run dev` to get a local instance of it running.
